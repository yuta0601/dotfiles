# DotFiles

## Supported OS
- macOS
- Ubuntu

## Installation
設定ファイルのシンボリックリンクを作成しデプロイします。


```sh
git clone https://github.com/yuta3003/dotfiles.git ~/dotfiles
cd ~/dotfiles/etc/scripts
./deploy.sh
```
